üõï Architecture

MonDictDB is an open-source Mon language dictionary database combined majorly from Nai Sac Lun's Mon-Burmese Dictionary and Mahajaroon's Committee' Mon-Thai Dictionary. We've implemented various automation algorithm using Python and MySQL to enrich, data cleanse, and compile into various end-products aimed to support most SQL database engine by using SQLite as baseline. This is also beneficial for developers who're looking for offline or in-app database.
