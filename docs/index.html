Docs & Demo

Interactive demo showing how to manage an open-source & centralized dictionary database of The Mon Language using SQL commands to create, read, update, and delete (CRUD).

This demo has also integrated SQL.js to run SQLite engine compiled into WebAssembly aka. WASM. (Can be used as offline database).
